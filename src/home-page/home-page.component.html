Overview

This project focuses on allowing developers to set up infinitely scrolling list pages with minimal setup/frustration.

A number of features are supported out of the box:

  • Infinitely scrolling list
  • Turning columns on/off
  • Reordering columns (active or inactive)
  • Setting column widths as a fraction of a page
  • Setting minimum/maximum widths for a column
  • Setting a column to a fixed width
  • Left/center/right aligning content
  • Adding/removing rows from an existing list
  • Allowing users to resize columns
  • Keep a column always visible on-screen
  • Show/hide columns at different screen widths
  • For data retrieval, choose between a basic array, or an asynchronous list, or a paginated list
